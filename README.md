# 🚀 Fluidic-Ops AI Business Intelligence Dashboard

## 📌 Project Overview
This project is an AI-powered Business Intelligence Dashboard built using Flask and Python. It helps in analyzing business data, generating insights, visualizing reports, and interacting through an intelligent chatbot.

---

## ⚙️ Features
- 📊 Data Visualization Dashboard
- 🤖 AI Chatbot for queries
- 📈 Business Data Analysis
- 📄 Automated PDF Report Generation
- 🌐 Web-based Interactive Interface

---

## 🧠 Tech Stack
- Python
- Flask
- Pandas
- HTML, CSS, JavaScript
- Plotly / Matplotlib

---

## 📂 Project Structure
- app.py → Main Flask application  
- templates/ → HTML files  
- static/ → CSS & JS files  
- data/ → Dataset files  

---

## 🚀 How to Run

```bash
pip install -r requirements.txt
python app.py
